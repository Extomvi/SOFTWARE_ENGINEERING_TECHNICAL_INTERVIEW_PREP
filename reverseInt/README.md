# REVERSE INT

## DESCRIPTION

Given a 32-bit signed integer, reverse digits of an integer.


### EXAMPLE 1

```BASH
Input: 123
Output: 321
```
### EXAMPLE 2

```BASH
Input: -123
Output: -321
```
### EXAMPLE 3

```BASH
Input: 120
Output: 21
```
